Demo Notes

Election Manager administers elections from start to finish. It gives the administrator tools to set up individual races and allows voters to vote. And it keeps track of the votes as they are cast.

Voter Login:
demovoter
demovoterpw
note: The "only vote once" control is turned off for the demo voter. Feel free to create a new voter to test this feature.

Admin Login:
demoadmin
demoadminpw

This app will only allow a voter to vote in races for their state, city and district (Miami, FL - 5 for the demo voter). However, once you are done voting and log out, you will see the results for all current races.
